FlaskNetScan - Network Scanner and Server Management Tool

About

FlaskNetScan is a comprehensive network scanning and server management solution built with Flask and modern web technologies. It provides a robust platform for managing multiple servers, monitoring system resources, and maintaining secure connections.

Key Features

  • Real-time network scanning
  • Server monitoring and management
  • Secure SSH connections
  • File transfer system
  • Resource monitoring
  • User authentication
  • Activity logging
  • Automated tasks

Technical Requirements

  • Python 3.9 or higher
  • Node.js 16 or higher
  • PostgreSQL 13+
  • Redis 6+
  • Modern web browser

Installation Guide

  1. Clone the repository
  2. Install Python dependencies: pip install -r requirements.txt
  3. Install Node dependencies: npm install
  4. Configure environment variables
  5. Run database migrations: flask db upgrade
  6. Start the server: python run.py

Security Features

  • SSL/TLS encryption
  • JWT authentication
  • IP whitelisting
  • Role-based access
  • Activity logging
  • Secure password handling
